# uAgent

Integração de agentes com API's de mobilidade, para encontrar estações de carregamentos e estacionamentos próximos.

## Instalação

```bash
python -m venv .venv
.venv\scripts\activate
pip install -r requirements.txt
```

## Baseamento para desenvolvimento

[fetch.ia](https://fetch.ai/docs/guides/integrations/fetch-mobility/walk-through)
